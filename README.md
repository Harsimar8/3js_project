# 🏛️ Immersive 3D Fine Art Gallery

A high-performance, interactive 3D art exhibition space engineered entirely in vanilla JavaScript utilizing **Three.js** and bundled via **Vite**. The application showcases seamless state transitions, environmental customization, dynamic lighting interactions, a custom collision system, and real-time bounding-box physics.

---

## 🚀 Getting Started

Follow these steps to set up the repository locally and spin up the development environment:

```bash
# 1. Navigate into the project folder
cd 3js

# 2. Install dependencies
npm install

# 3. Spin up the Vite local development server
npm run dev
